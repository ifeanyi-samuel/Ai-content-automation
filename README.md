# Ai-content-automation workflow
 AI-powered content automation workflow built with n8n.

## Overview

![workflow overview](ai%20content.png)

An AI-powered content automation workflow built with n8n that discovers trending TikTok topics, generates weekly content ideas, creates AI-written scripts, and delivers the results automatically.

## Features
- Runs automatically on a weekly schedule
- Fetches trending TikTok content using an API
- Generates 10+ content ideas with AI
- Creates ready-to-use content scripts
- Formats the output using JavaScript
- Stores results in Google Drive
- Sends notifications through Telegram

## Technologies Used
- n8n
- OpenAI
- HTTP Request
- JavaScript
- Google Drive
- Telegram Bot
